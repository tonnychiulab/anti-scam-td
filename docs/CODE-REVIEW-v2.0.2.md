# 防詐迷宮 v2.0.2 修補與驗證報告

- 版本：v2.0.2
- 日期：2026-07-13
- 前次報告：`docs/CODE-REVIEW-v2.0.1.md`
- 驗證方式：原始碼複查、Node 回歸測試、多輪重複測試、本機 PC 瀏覽器互動測試
- 結論：v2.0.1 報告所列 5 項殘餘問題均已修補；本次測試範圍內沒有發現新的可重現 bug

## 一、修補摘要

v2.0.2 針對 v2.0.1 複審發現的 PC 操作、鍵盤焦點、無障礙、多語提示、視覺對比與大霧範圍問題進行修補，並同步更新應用版本及 Service Worker 快取名稱，避免玩家繼續取得舊檔。

本次沒有改動關卡數、塔的價格、敵人數值、地圖種子或排行榜規則。

## 二、問題修補對照

| 編號 | v2.0.1 問題 | v2.0.2 結果 | 驗證 |

## 三、程式複查重點

### 焦點與滑鼠

- `game.js:1208-1209` 將棋盤聚焦集中為單次 `focusBoard()`。
- `game.js:1578` 忽略右鍵、中鍵及其他非主要按鍵。
- `game.js:1639` 僅在遊戲 canvas 阻止右鍵選單，頁面其他區域仍保留瀏覽器原生右鍵功能。
- `game.js:1726,1736,1979` 只在選塔、選支援及開始遊戲等明確操作後把焦點交回棋盤。

### 多語與無障礙

- `game.js:1211-1229` 集中更新 HUD 控制的 `title`、`aria-label` 與 `aria-pressed`。
- `game.js:1881-1939` 切換語言時同步更新 HUD、塔、支援按鈕與語言選擇狀態。
- `i18n.js` 的中文、英文、印尼文、越南文均含 5 個 HUD 標籤、控制文字、13 個塔提示及 3 個支援提示。

### 遊戲規則與視覺

- `game.js:268-276` 與 `game.js:343-350` 的被動 buff／轉化距離使用事件射程倍率。
- `style.css:63-64,133` 的綠色、紅色與支援瞄準按鈕改為深色文字。
- `game.js`、`sw.js`、`package.json`、`index.html`、四語版本文字與 README 均同步為 v2.0.2。

## 五、PC 瀏覽器實測

### 焦點

1. 開始遊戲後點擊速度按鈕，速度由 1× 切換為 2×。
2. 等待 350ms 後，焦點仍為 `#btnSpeed`，沒有被 animation loop 拉回 canvas。
3. 主動選擇「超商工讀生」後，焦點才依設計回到 `#cv`。

### 右鍵

1. 新遊戲點數為 120，選擇價格 40 的「超商工讀生」。
2. 在「第 1 欄、第 1 列：草地，可建造」以滑鼠右鍵點擊，點數仍為 120，沒有建塔。
3. 對同一座標改用左鍵，點數正常變為 80，證明左鍵建塔沒有被修補破壞。

### 多語與快取

- 首頁及遊戲狀態均顯示 v2.0.2。
- 第二次載入顯示 `CACHE 100% (3/3)`，新 Service Worker 快取可正常使用。
- 英文介面的 5 個 HUD tooltip、速度／暫停／靜音標籤、塔提示與支援提示均實際更新為英文。
- 三個測試分頁均沒有 console error。

## 六、限制與發行判定

本次結果表示在已檢查的原始碼、28 項自動回歸與 PC 瀏覽器操作範圍內，沒有已知可重現 bug；不等同保證任何裝置與極端條件下絕對零缺陷。

仍建議正式發佈前或發佈後補做：

- iPhone Safari 與 Android Chrome 真機測試。
- VoiceOver／NVDA 等實際螢幕閱讀器流程。
- 正式 HTTPS 環境的完整離線重啟與更新測試。
- 長時間多關卡、背景切換及低階裝置效能測試。

**發行判定：v2.0.2 可作為本輪 PC／無障礙問題的修補版候選。**
