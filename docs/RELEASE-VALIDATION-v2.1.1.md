# v2.1.1 修補與驗證紀錄

- 版本：`v2.1.1`
- 驗證日期：2026-07-15
- 分支：`agent/v2.1-mobile-ux`
- 範圍：手機手勢、暫停蓄力、無障礙偏好、建塔流程、PWA 更新

## 修補摘要

1. 放大棋盤後，單指拖曳可平移；雙指手勢開始時會取消待處理點擊並安全關閉建塔預覽。
2. 支援技能暫時接管建塔介面，取消瞄準或蓄力時會回復原本的塔種與預覽位置；成功施放後才清除建塔狀態。
3. 強光手電筒的 500ms 蓄力改以遊戲時間計算；遊戲暫停或開啟「更多」造成暫停時，蓄力不會在背景完成。
4. 靜音偏好寫入本機設定；減少動態或最低畫質模式不再觸發震動。
5. 建塔面板補上選取狀態、焦點回復、多語取消名稱，以及餘額不足、格子占用、塔未解鎖等可朗讀錯誤提示。
6. 手機「更多」面板可開啟玩法說明。
7. Service Worker 核心文件、程式與樣式改為 network-first，離線時才回退快取；新版等待中或接管後會顯示「立即更新」。
8. 遊戲、快取、四語版本字樣與套件版本統一為 `2.1.1`。
9. PC 以滑鼠選擇下方塔種時，不再於棋盤左上角誤顯示鍵盤格線；Tab／方向鍵操作仍保留焦點提示。

- 加入 PC 焦點框回歸檢查後：`40/40` 通過。
- JavaScript 語法檢查：`game.js`、`i18n.js`、`sw.js` 全部通過。
- 差異格式檢查：通過；僅有工作區既有的 LF/CRLF 提示，沒有空白錯誤。

新增或擴充的回歸範圍包括：

- 手機單指平移、雙指手勢與建塔預覽關閉。
- 強光蓄力在暫停期間凍結，恢復後才繼續。
- 減少動態／最低畫質停用震動。
- 靜音設定的舊資料遷移、保存與重載。
- 支援取消後回復建塔流程、錯誤提示與焦點語意。
- PWA 核心資源優先取網路，斷線時使用快取。
- 滑鼠造成的棋盤焦點不繪製鍵盤格線，鍵盤焦點仍會繪製。

## 瀏覽器操作驗證

以本機 v2.1.1 頁面進行桌面瀏覽器操作，並暫時套用 `390 × 844` 視窗尺寸降低畫布負載：

- 起始畫面與遊戲內版本均顯示 `v2.1.1`。
- 可正常開始遊戲、暫停與繼續。
- 在遊戲棋盤按滑鼠右鍵後，頁面維持原狀，沒有出現遊戲內選單或誤觸操作。
- 切換靜音後重新載入並再次開始遊戲，仍維持靜音；驗證後已還原為開啟音效。
- 建塔面板具有 `aria-describedby="bmPreview bmError"`，錯誤區為 `role="status"` 與 `aria-live="polite"`。
- 「更多」玩法說明與「立即更新」按鈕的中文文字均已載入。

此外，舊測試來源仍可重現 v2.0.2 Service Worker 提供舊快取頁面的情形；v2.1.1 已針對後續版本改用核心資源 network-first 與明確更新流程。舊版首次升級仍可能需要一次重新整理，之後將由新策略接管。
